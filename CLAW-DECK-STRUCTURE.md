# CLAW DECK - LAYOUT STRUCTURE

**THIS IS THE STRUCTURE. NEVER CHANGE IT UNLESS USER ASKS.**

## Layout (Top to Bottom)

1. **HEADER** - Logo + system pills (Gateway, Browser, GitHub) + status badge
2. **DATETIME BAR** - Greeting (left) + Time (right)
3. **ZEN QUOTE** - Full width quote section
4. **TEAMS** - Horizontal grid of team cards (click to show details)
5. **TEAM DETAILS** - Expandable panel showing project details for selected team
6. **PRIORITIES** - Numbered list (P1, P2, P3)
7. **FINANCE** - Income/expenses grid + Canada Fund progress bar
8. **TO-DO** - Two columns: TODO + DONE
9. **LINKS** - Bottom, horizontal wrap of project links

## Features to Always Keep

- Click team card → shows details panel
- ⚙️ Settings button → edit Canada Fund target/current
- Local storage → saves settings
- Live clock
- Greeting changes based on time of day

## Never Change Without Permission

- Order of sections
- Basic layout structure
- Team dropdown behavior
